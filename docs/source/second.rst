welcome to pointillism!
=======================

Diagram rendering as a Service.

`pointillism` works with virtually all the services you use.
`Real-Time` rendered diagrams won't get out-dated.

* `github <https://github.com/trevorgrayson/pointillism#embedding-dot-graphs-in-github>`_
* wikis (`MediaWiki <https://www.mediawiki.org/wiki/MediaWiki>`_, \
  `Confluence <https://www.atlassian.com/software/confluence>`_, ...)
* `second <./second>`_

Technologies
------------

Many technologies render `pointillism` compatable graph files.

* `terraform <https://www.terraform.io/docs/commands/graph.html>`_
